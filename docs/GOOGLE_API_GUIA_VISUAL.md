╔════════════════════════════════════════════════════════════════╗
║     GUIA VISUAL: Ativando Google Sheets API                    ║
╚════════════════════════════════════════════════════════════════╝

Se você só vê "Service Usage API" e "Service Management API",
é porque está no lugar errado! Siga estes passos VISUAIS:

═══════════════════════════════════════════════════════════════
PASSO 1: ACESSAR BIBLIOTECA DE APIs
═══════════════════════════════════════════════════════════════

1. Vá para: https://console.cloud.google.com/
2. No MENU LATERAL ESQUERDO, clique em:
   📁 "APIs e Serviços" → 🗂️ "Biblioteca"

   (NÃO clique em "Credenciais" ainda!)

═══════════════════════════════════════════════════════════════
PASSO 2: PROCURAR GOOGLE SHEETS API
═══════════════════════════════════════════════════════════════

Na barra de busca da Biblioteca, digite: "sheets"

Encontre: "Google Sheets API"
📄 Descrição: "Reads and writes Google Sheets."

Clique nela → Botão "ATIVAR" (azul)

═══════════════════════════════════════════════════════════════
PASSO 3: PROCURAR GOOGLE DRIVE API
═══════════════════════════════════════════════════════════════

Na barra de busca, digite: "drive"

Encontre: "Google Drive API"
📄 Descrição: "Manages files in Google Drive."

Clique nela → Botão "ATIVAR" (azul)

═══════════════════════════════════════════════════════════════
PASSO 4: CRIAR SERVICE ACCOUNT
═══════════════════════════════════════════════════════════════

Agora sim, vá para:
📁 "APIs e Serviços" → 🔑 "Credenciais"

Clique: "+ CRIAR CREDENCIAIS" → "Chave de conta de serviço"

═══════════════════════════════════════════════════════════════
VERIFICANDO SE ATIVOU CERTO:
═══════════════════════════════════════════════════════════════

Volte para "Biblioteca" e procure novamente por "sheets":

✅ Se aparecer "Google Sheets API" com botão "GERENCIAR" = ATIVADA!
✅ Se aparecer "Google Sheets API" com botão "ATIVAR" = NÃO ATIVADA!

═══════════════════════════════════════════════════════════════
RESUMO VISUAL:
═══════════════════════════════════════════════════════════════

1. 🏠 console.cloud.google.com
2. 📁 APIs e Serviços → 🗂️ Biblioteca
3. 🔍 Buscar "Google Sheets API" → 🔵 ATIVAR
4. 🔍 Buscar "Google Drive API" → 🔵 ATIVAR
5. 📁 APIs e Serviços → 🔑 Credenciais
6. ➕ Criar Service Account
